# Open Source Quiz (CS200)
creaters
Noah pinkley
Kacie Inscoe

Overview
This project is an interactive Twine story built in Harlowe 3.3.9. It teaches users about open source software through explanations, examples, and a short quiz at the end.

What the Project Does
Users explore different passages that explain:
- What open source software is
- The history of open source
- Why open source is important
- Benefits and drawbacks
- Real-world examples of open source tools

At the end, users take a 5-question quiz to test their knowledge.

Quiz System
- Each correct answer adds to the score
- Final score is shown at the end:
  - 5/5 → Perfect score
  - 3–4/5 → Good understanding
  - 0–2/5 → Needs review

Features
- Interactive branching story format
- Multiple-choice quiz
- Score tracking using variables
- Immediate feedback for answers
- Final results screen

Examples Included
- Linux
- Git
- VS Code
- Audacity
- Android

Tools Used
- Twine
- CSS styling (StoryStylesheet)

How to Run
1. Open Twine
2. Import the project or paste the code
3. Click Play
4. Navigate through the story and complete the quiz
